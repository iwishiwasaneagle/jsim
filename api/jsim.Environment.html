abstract step(pa: jsim.Meta.Action.Action) → Tuple[jsim.Meta.State.State, float][source]

Must be provided by the user for their specialized environment. Called once every simulation step and causes the environment to undergo a transition from its current state to a next state dependent on the action taken by pa.

Note

If the environment transitions into a terminal state, the returned state will have a special value of 0

Parameters

pa (Action) – The action to be taken by the Agent

Returns

  • state - The next state due to action pa

  • reward - The payoff for the state transition due to action pa

Return type

Tuple[State, float]
