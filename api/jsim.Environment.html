jsim.Environment.Environment module¶

The environment defines the problem to be solved. It determines the dynamics of the environment, the rewards, and the trial terminations.

abstract start_trial() → jsim.Meta.Sensation.Sensation[source]¶

This function must be provided by the user for their specialized environment. Called at the beginning of a new trial and should perform any needed initialization of the environment to prepare for a new trial.

abstract step(pa: jsim.Meta.Action.Action) → Tuple[jsim.Meta.Sensation.Sensation, float][source]¶

Must be provided by the user for their specialized environment. Called once every simulation step and causes the environment to undergo a transition from its current state to a next state dependent on the action taken by pa.

Note

If the environment transitions into a terminal state, the returned sensation will have a special value of 0

Parameters

pa (Action) – The action to be taken by the Agent

Returns

  • sensation - The next sensation due to action pa

  • reward - The payoff for the state transition due to action pa

Return type

Tuple[Sensation, float]
